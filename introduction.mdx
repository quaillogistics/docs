# DotLookup API

Free, open REST API for FMCSA carrier data. No API key required for basic lookups.

## Base URL

```
https://api.dotlookup.dev/v1
```

## Quick Start

Look up any carrier by DOT number — no signup needed:

```bash
curl https://api.dotlookup.dev/v1/carriers/2233855
```

## Data freshness

DotLookup re-runs the FMCSA ETL pipeline daily. [`GET /v1/meta`](/endpoints/meta) exposes `data_freshness` (the timestamp of the most recent run), plus `risk_history_horizon` and `etl_changes_horizon` for the time-anchored endpoints.
