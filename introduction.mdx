# DotLookup API

Free, open REST API for FMCSA carrier data. No API key required for basic lookups.

## Base URL

```
https://api.dotlookup.dev/v1
```

## Quick Start

Look up any carrier by DOT number — no signup needed:

```bash
curl https://api.dotlookup.dev/v1/carriers/2233855
```

## Authentication

Most endpoints work without authentication. Include an API key via the `X-API-Key` header for higher rate limits and access to the [bulk lookup](/endpoints/bulk-lookup) endpoint.
